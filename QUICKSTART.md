# Quick Start Guide

## Install & Run in 2 Minutes

### 1. Install Dependencies
```bash
npm install
```

### 2. Configure Environment
```bash
cp .env.example .env
# Edit .env with your API URL
```

### 3. Start Development Server
```bash
npm run dev
```

Visit [http://localhost:5173](http://localhost:5173)
