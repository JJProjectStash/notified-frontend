## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/notified-frontend.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Run tests: `npm run lint && npm run format:check`
6. Commit: `git commit -m 'Add some feature'`
7. Push: `git push origin feature/your-feature-name`
8. Open a Pull Request

## Code Standards

- Use TypeScript for all new code
- Follow the existing code style (enforced by ESLint/Prettier)
- Write meaningful commit messages
- Add comments for complex logic
- Update README if needed

## Pull Request Process

1. Update documentation if you've changed functionality
2. Ensure all lint checks pass
3. Get approval from at least one maintainer
4. Squash commits before merging

## Questions?

Open an issue or reach out to the maintainers.
